# EB 自动化启动器（Windows 免安装版）使用说明

## 1. 文件说明

- `EB Daily Exporter Portable.exe`：主程序（免安装，双击即可运行）
- `英文黑道每日数据模板.xlsx`：模板文件（程序会基于它生成每日结果）

请确保模板文件与程序在同一目录下。

## 2. 启动与运行

1. 双击 `EB Daily Exporter Portable.exe`
2. 在界面填写：
   - 账号
   - 密码
   - 日期（支持 `4.28` / `4.28,4.30` / `4.28-4.30`）
   - 搜索后等待秒数（默认 35）
   - 浏览器模式（可见 / 后台）
3. 点击“开始执行日报脚本”
4. 运行结束后查看输出目录

## 3. 输出结果

每个日期会生成一个文件夹，例如：

- `每日数据4.28`
- `每日数据4.29`

每个文件夹内包含 3 个文件：

- `CDP_运营报表_4.28_全地区.xlsx`
- `CDP_运营报表_4.28_美国.xlsx`
- `英文黑道每日数据模板_4.28.xlsx`

说明：
- 两个 `CDP_...` 文件是原始导出数据
- `英文黑道每日数据模板_4.28.xlsx` 是合并后的结果文件（会写入指定 sheet）

## 4. 停止脚本

- 点击界面的“停止脚本”可中止当前任务
- 直接关闭窗口也会触发停止并退出程序

## 5. 浏览器说明

程序按以下顺序尝试启动浏览器：

1. 包内浏览器（若存在）
2. 系统 Microsoft Edge
3. 系统 Google Chrome

所以一般不需要用户手动执行 `npx playwright install`。

## 6. 常见问题

### Q1: 点击开始后立即报“缺少账号密码”
- 请确认账号和密码不为空

### Q2: 日期格式报错
- 正确示例：`4.28`、`4.28,4.30`、`4.28-4.30`
- 不支持 `2026-04-28` 这种格式

### Q3: 没有导出文件
- 提高“搜索后等待秒数”（例如改为 60）
- 检查目标网站是否正常打开并已登录

### Q4: 浏览器启动失败
- 先确认本机至少安装 Edge 或 Chrome 其中之一
- 若公司策略限制浏览器启动，请联系 IT 放行

## 7. 版本与更新建议

- 每次拿到新版本，建议直接替换整个程序目录
- 不要只替换单个 `.exe`，避免与旧脚本/模板不匹配
